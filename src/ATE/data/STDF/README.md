# The ATE.org STDF library

## Introduction

#### This library is **not** intended to be the 'fastes in the world'! 



#### It is also **not** set up as just a parser! It can read **and** write!


## Support
 - Endians: Little & Big
 - Versions & Extensions:
   - V3 : standard, +
   - V4 : standard, V4-2007, Memory:2010.1
 - Modes: read & write
 - compressions:
   - gzip
   - xz
   - bz2
