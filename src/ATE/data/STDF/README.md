# The ATE.org STDF library

## Introduction

In writing STDF parsers


## Support
 - Endians: Little & Big
 - Versions & Extensions:
   - V3 : standard, +
   - V4 : standard, V4-2007, Memory:2010.1
 - Modes: read & write
 - compressions:
   - gzip
   - xz
   - bz2
